# Lacleo-Foods-Website
LaCleo Foods is a B2B food supply platform designed to connect businesses such as restaurants, retailers, and food service providers with bulk food products and ingredients. The platform focuses on streamlining procurement by offering a range of staple and packaged food items for commercial use. As a relatively new and emerging website, it provides secure access but has limited public presence and verification, making it suitable for exploration while encouraging cautious engagement for business transactions.

ReadMe:
1. Open Visual Studio
2. Open the my-catering-website file
3. Open a new terminal
4. Create a new venv folder using
   pip install venv venv
(Make sure you're downloading the venv file in the my-catering-website folder only)
Install all pip libraries using
pip install flask requests python-dotenv textblob transformers torch reportlab
7. Run venv environment using
.\venv\Scripts\activate
8. Run the program using 
python app.py
